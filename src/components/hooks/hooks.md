When to use what (simple)

    Small state → useState

    Side effects → useEffect

    Shared data → useContext

    DOM access → useRef

    Performance → useMemo, useCallback

    Complex state → useReducer

Hooks allow Functional Component to use State & LifeCycle Feature without using Class Component.


Real Project Example: Task Manager (Mini App)

Features:

    Add tasks

    Persist data

    Optimize performance

    Access DOM

    Share state

Hooks we’ll use:

    useState

    useEffect

    useRef

    useMemo

    useCallback
